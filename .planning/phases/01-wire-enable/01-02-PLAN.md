**INTEG-04: Wire update_bankroll() into continuous mode**

Two trigger paths per user decision: timer-based (every 5 minutes) AND post-trade (immediate after any trade execution).

1. Add timer initialization alongside existing `_last_snapshot_time` in the continuous loop setup area. Find where `_last_snapshot_time = 0.0` is initialized (around line ~730-740) and add:

```python
_last_bankroll_refresh = 0.0
_bankroll_refresh_interval = 300.0  # 5 minutes
```

2. Add timer-based refresh inside the main scan loop, AFTER the execution block (after opportunities have been processed and any trades executed). Place it near the existing snapshot recording logic:

```python
# Timer-based bankroll refresh (every 5 minutes)
_now = time.time()
if _now - _last_bankroll_refresh >= _bankroll_refresh_interval:
    try:
        balances = executor._fetch_balances("Cross")
        if balances and executor.position_sizer:
            total = sum(v for v in balances.values() if isinstance(v, (int, float)))
            if total > 0:
                executor.position_sizer.update_bankroll(total)
                logger.info("Bankroll refreshed: $%.2f across %d platforms", total, len(balances))
        _last_bankroll_refresh = _now
    except Exception as exc:
        logger.debug("Bankroll refresh failed: %s", exc)
        _last_bankroll_refresh = _now  # Don't retry immediately on failure
```

3. Add post-trade refresh right after `executor.execute(opp)` returns True. Find the execution block in the continuous loop where `executor.execute(opp)` is called (search for `executor.execute(` in continuous.py). After a successful execution:

```python
if executor.execute(opp):
    executed += 1
    # Immediate bankroll refresh after trade (per user decision)
    try:
        balances = executor._fetch_balances("Cross")
        if balances and executor.position_sizer:
            total = sum(v for v in balances.values() if isinstance(v, (int, float)))
            if total > 0:
                executor.position_sizer.update_bankroll(total)
    except Exception as exc:
        logger.debug("Post-trade bankroll refresh failed: %s", exc)
```

CRITICAL GUARDS:
- Always check `executor.position_sizer` is not None before calling `update_bankroll()` — it's None when position sizing is not configured
- Always check `balances` is not None and `total > 0` before updating
- Catch all exceptions — bankroll refresh failure must never break the scan loop
- `_fetch_balances("Cross")` fetches ALL 8 platforms — this is the right call for total capital picture
- `time` module is already imported in continuous.py
